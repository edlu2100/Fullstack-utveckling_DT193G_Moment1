Vad är Tailwind?

Tailwind CSS är ett populärt CSS-ramverk som används för att utveckla webbgränssnitt. Det skiljer sig från många andra CSS-ramverk genom att det förespråkar en "utility-first" metodik, vilket innebär att det tillhandahåller ett omfattande uppsättning CSS-klasser som kan användas direkt i HTML-markupen för att stila webbelement.

Vad tycker jag om Tailwind?

Jag tycker att Tailwind är riktigt bra. Den har funktioner som jag saknade på Bootstrap. Det är lätt att sätta avstånd och bredd. Som jag skrev om Bootstrap så saknade jag större marginaler, vilket det finns här. Här går det långt över vad man behöver. Det negativa är att det krävs lite att ladda ner via npm men sen fungerar det hur bra som hellst. Kommer verkligen att använda det i framtiden. Känns som att allting föll på plats. Någonting man saknat under alla projekt vi gjort. Känns som att det börjar bli på riktigt, att det blir mer avancerat och snyggt.
