Vad är Bootstrap?

Bootstrap är ett gratis ramverk för frontend-utveckling. Bootstrap har öppen källkod, vilket betyder att alla kan ta del av det och skapa webbplatser och appar. Bootrsp hjälper att skapa responsiva webbplatser med en uppsättning av malldesigner som är färdigskapade. Bootstrap innehåller grunderna och utvecklaren behöver endast infoga koden för att ta del av funktionaliteten. De är byggt för att underlätta och effektivisera kodningen i HTML, CSS och JavaScript.

Vad tycker jag om Bootstrap?

Jag anser att Bootstrap är ett bra ramverk. Jag gillar verkligen hur lätt det var att göra formulär och tabeller. Jag tycker även att det var väldigt lätt att rada upp saker horisontellt och ändra storlek beroende på skärmstorleken. Däremot tycker jag att de saknas vissa egenskaper. Jag tycker det slulle vara lättare att sätta width på formuläret och tabellen. Jag testade med col men det fungerade imte- Det kanske finns nån funktionallitet som jag missat men jag fick istället style den med css. Synd då jag anser att det var lätt att jobba med col-#.Sen anser jag att att det hade kunnat finnas några extra steg när man sätter marginaler, inte bara till 5. Den gör inte så stor skillnad så jag ahde uppskattat om det fanns till 10. Annars tycker jag Bootstrap var enkelt och snabbt att arbeta med.
