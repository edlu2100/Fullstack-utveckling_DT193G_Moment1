CSS-Ramverk

I den här uppgiften, Moment 1 - CSS Ramverk, ska två olika undersidor med olika ramverk skapas. Det ska även finnas en startsida med innehåll om uppgiften samt en sida om hur man publicerat uppgiften. På undersidorna ska det finnas information om ramverket plus länk till deras sida, en tabell med minst fyra fält och fem rader data, samt ett formulär.

I den här uppgiften har jag valt att använda mig av Bootstrap samt Tailwind.

Bootstrap

Bootstrap är de vanligaste ramverket fär frontend-utveckling. Bootstrap är ett gratis ramverk vilket effektiviserar kodningen i HTML, JavaScript samt CSS.

Tailwind

Tailwind CSS är ett populärt CSS-ramverk som används för att utveckla webbgränssnitt. Ramverket tillhandahåller CSS-klasser som kan användas direkt i HTML-markupen för att stila webbelement.
